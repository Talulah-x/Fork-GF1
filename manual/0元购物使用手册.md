# 0元购_无核心使用手册

⚠️打开记录缩放

**如何运行**:

a.进入卷积核—打捞关卡【莫比乌斯带上的逃亡*普通难度】将 地图缩放到最小 且 向右上拉不动 为止，部署重装后开始运行

b.直接从主界面/战斗—常驻活动——卷积核开始运行

**关卡收益**:

消耗：无

粗略收益：

1. 人力：700/h
2. 弹药：700/h
3. 口粮：550/h
4. 零件：200/h(50取整，且包括资源自回)
